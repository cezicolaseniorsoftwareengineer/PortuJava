Onde a árvore começa?

Na raiz do produto. Para este monorepo: backend/src/main contém o backend, frontend/src contém a interface, contracts contém contratos versionados, infrastructure contém entrega e operação, e docs preserva decisões. Não comece por data: banco de dados é uma dependência persistente, não a raiz da arquitetura.
